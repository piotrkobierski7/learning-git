NODE.JS

is a platform for running JavaScript code. The platform provides an API, whose widest (but not the only) application is the creation of websites based on event-driven programming. In addition to the mere ability to run JavaScript in a non-browser environment, the platform offers many modules that form a powerful set of development tools.

Bootstrap

CSS library, developed by Twitter programmers, licensed by MIT [1]. It contains a set of useful tools to help you create a graphical interface of websites and web applications. It is based mainly on ready-made HTML and CSS solutions (compiled from Less files) and can be used, among others, for styling elements such as texts, forms, buttons, charts, navigations and other components displayed on the page. The library also uses JavaScript.

React.js

JavaScript programming language library, which is used to create graphical interfaces of web applications. It was created by Jordan Walke, a Facebook programmer, and inspired by the extension of PHP - XHP. The main features distinguishing the React.js library are the virtual DOM (Document Object Model, in Polish Object Model Document). React stores the entire DOM of the application in memory, after changing the state searches for differences between the virtual and real DOM and updates the changes.
